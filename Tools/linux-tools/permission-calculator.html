Special Permissions

SetUID (SUID)

When set, the file executes with the privileges of the file owner, regardless of who runs it.

4000

SetGID (SGID)

For files: execute with group privileges. For directories: new files inherit directory's group.

2000

Sticky Bit

When set on a directory, only the file owner can delete or rename files within that directory.

1000

Octal Value Binary Permissions Description
0 000 000 --- No permissions

Symbol Meaning Octal Description
r Read 4 Read file or list directory
w Write 2 Modify file or directory contents
x Execute 1 Execute file or enter directory
s SetUID/SetGID 4/2 Special execute with owner/group privileges
t Sticky 1 Restricted deletion in directories
- None 0 Permission not granted
Use Case Octal Symbolic chmod Command
Secure configuration file 600 rw------- chmod 600 config.conf
Public web directory 755 rwxr-xr-x chmod 755 public_html/
Shared group directory 775 rwxrwxr-x chmod 775 shared/
Read-only documentation 444 r--r--r-- chmod 444 README.md
SUID root binary 4755 rwsr-xr-x chmod 4755 /usr/bin/passwd
Sticky temp directory 1777 rwxrwxrwt chmod 1777 /tmp
Special Bit Octal Symbol Effect
SetUID (SUID) 4000 s in user execute File runs with owner's privileges
