How to Check Windows Versionকিভাবে Windows ভার্সন চেক করবেন

Winver CommandWinver কমান্ড

Easyসহজ
1

Open Run Dialog

Run ডায়ালগ খুলুন

Press Windows Key + R

টিপুন Windows কী + R

2

Type Winver

টাইপ করুন Winver

Type the following command:

নিম্নলিখিত কমান্ড টাইপ করুন:

winver
3

Press Enter

Enter টিপুন

A dialog will show your Windows version and build number

1

সেটিংস খুলুন

টিপুন Windows কী + I বা Start মেনু → সেটিংস খুলুন

Quick Tips & Informationদ্রুত টিপস ও তথ্য

Why Check Windows Version?

কেন Windows ভার্সন চেক করবেন?

Knowing your Windows version helps ensure security updates, software compatibility, and system stability.

আপনার Windows ভার্সন জানা নিরাপত্তা আপডেট, সফ্টওয়্যার সামঞ্জস্য এবং সিস্টেম স্থিতিশীলতা নিশ্চিত করতে সাহায্য করে।

Version Numbering Explained

ভার্সন নম্বরিং ব্যাখ্যা

Windows versions use Year (YY) and Half (H) format: 22H2 means second half of 2022. Build numbers are incremental updates.

Windows ভার্সনগুলি বছর (YY) এবং অর্ধ (H) ফরম্যাট ব্যবহার করে: 22H2 মানে 2022 সালের দ্বিতীয়ার্ধ। বিল্ড নম্বরগুলি ক্রমবর্ধমান আপডেট।

Security Warning

নিরাপত্তা সতর্কতা

Unsupported Windows versions (like Windows 7) don't receive security updates, making them vulnerable to attacks.

অসমর্থিত Windows ভার্সনগুলি (যেমন Windows 7) নিরাপত্তা আপডেট পায় না, যা এগুলিকে আক্রমণের জন্য ঝুঁকিপূর্ণ করে তোলে।
