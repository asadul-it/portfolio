Additional Information

About Bangladesh Postal Codes

Format: 4-digit numbers (e.g., Dhaka GPO - 1000, Chittagong GPO - 4000)

Coverage: All 64 districts, 492 upazilas/thanas, and 4,571 unions across Bangladesh

Structure: District Code + Upazila Code + Union Code

Usage: Essential for courier services, online shopping, and official correspondence

Updated: Regular updates from Bangladesh Post Office official sources

How to Use This Tool

Method 1: Select district, then upazila, then union for precise results

Method 2: Search directly by union, thana, or post office name

Method 3: Use quick search for any location name

Copy Codes: Click the copy button next to any postal code

Browse: Use pagination to navigate through large result sets

Important Notes

Postal code information is for reference purposes. For official use, please verify with Bangladesh Post Office.

Some remote or newly established areas may have different postal codes.

Major cities like Dhaka and Chittagong have multiple postal codes for different areas.

Always double-check postal codes before sending important mail or documents.
