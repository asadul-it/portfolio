Bangladesh NID Information Checker

Extract birth date, age, and gender information from Bangladesh National ID (NID) card number. This tool decodes the 10, 13, or 17 digit NID number to reveal the encoded information.

NID Number Format Information

  • 10 Digit NID: Old format voter ID numbers
  • 13 Digit NID: New format with birth date (YYYYMMDDXXXXX)
  • 17 Digit NID: Smart NID with extended information
  • Birth Date: First 8 digits represent birth date (YYYYMMDD)
  • Gender: Determined from the last digit before check digit
