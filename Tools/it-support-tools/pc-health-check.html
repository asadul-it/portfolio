Common Issues & Recommendations

Recommendations:

  • Regularly run Windows Update
  • Keep antivirus software updated
  • Disable unnecessary startup programs
  • Run Disk Cleanup regularly
  • Keep at least 15-20% free space on hard drive
  • Clear browser cache regularly
  • Uninstall unused software

Quick Solutions

Slow Computer:
  • Open Task Manager (Ctrl+Shift+Esc) and close high CPU/Memory programs
  • Run Disk Cleanup (Windows + R → cleanmgr)
  • Reduce startup programs
Slow Internet:
  • Restart your router
  • Clear browser cache
  • Disable unnecessary browser extensions
Full Hard Drive:
  • Run Disk Cleanup
  • Clean Downloads folder
  • Uninstall unused programs
  • Delete temporary files
